# Movie Ticket Management System
A GUI-based Movie Ticket Management System developed using Python (Tkinter).

## Features
- User and Admin Login System
- Ticket Cancellation Request
- Seat Change Request
- Show Time Change Request
- FIFO Queue-based Request Processing
- Request History Tracking

## Concepts Used
- Data Structures (Queue - FIFO)
- Python Lists
- Tkinter GUI
- Basic Authentication Logic

## How to Run
1. Install Python
2. Run the file:
   python movie_ticket_system.py
